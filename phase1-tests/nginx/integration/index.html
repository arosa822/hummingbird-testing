About This Site

This is a demonstration static website served by a Hummingbird nginx container.

It shows that the nginx image can be used as a base for hosting real websites with custom configuration.

Features Demonstrated

  • Custom nginx configuration
  • Static file serving (HTML, CSS, JS)
  • Health check endpoint
  • JSON API endpoint
  • Security headers
  • Gzip compression

API Endpoints

GET /health - Health check endpoint
GET /api/status - Status endpoint (JSON)
